# Save Design System v2.6 (Pastel Revolution)

Este sistema de diseño representa la evolución de **Save** hacia una estética de **Psychological Fintech** de alto impacto, basada en una paleta pastel que promueve la calma, el control y la elegancia.

---

## 🎨 Identidad Core (Pastel Sage)

Hemos adoptado una paleta cromática inspirada en tonos orgánicos y suaves, optimizada para interfaces modernas y limpias.

### Colores Principales
- **Primary (Mint/Teal):** `#47ADA2` - El color del control y la frescura.
- **Secondary (Green):** `#B9E2A2` - Representa el crecimiento y la fluidez.
- **Tertiary (Salmon):** `#F0927B` - Acentos de calidez y acción.
- **Accents:** 
  - Lavender (#D2A9D1) para alertas y jerarquía secundaria.
  - Emerald (#8AD6CE) para confirmaciones y éxito.

### Gradientes de Marca
- **Brand Gradient:** `[#8AD6CE, #B9E2A2, #D2A9D1]` (Usado en FAB y acciones principales).
- **Glass Effect:** Superficies al 75% de opacidad con bordes sutiles de luz blanca.

---

## 📊 Visualización de Datos (Minimalist Power)

El sistema v2.6 introduce un lenguaje visual para estadísticas que prioriza la velocidad de información:

1. **Barras Estilizadas:** Uso de formas "píldora" con iluminación interna.
2. **Segmentación de Capital:** Barras horizontales ultra-finas para distribución.
3. **Indicadores de Tendencia:** Tipografía `caption` resaltada con colores de éxito/error pastel.

---

## 📐 Sistema de Espaciado (Grid 8px)

| Token | Size | Uso |
| :--- | :--- | :--- |
| **xs** | `4px` | Ajustes finos y decoraciones. |
| **sm** | `8px` | Espaciado interno de inputs. |
| **md** | `16px` | Padding estándar de tarjetas. |
| **lg** | `24px` | Margen interno de secciones. |
| **xl** | `32px` | Márgenes laterales de pantalla. |
| **xxl** | `48px` | Separación de grandes bloques. |

---

## 🧩 Tipografía (Visual Rhythm)

Utilizamos **Outfit** para titulares (Headline) e **Inter** para cuerpo de texto (Body).

- **h1:** `32px` / Black (900) - Títulos de sección.
- **title:** `17px` / ExtraBold (800) - Nombres de tarjetas.
- **body:** `14px` / Medium (500) - Información general.
- **caption:** `11px` / Bold (700) - Etiquetas y micro-data.

---

## ✨ Sombras Premium (Tinted Elevation)

Las sombras ya no son grises neutros; ahora tienen un **ligero matiz Teal** para integrarse orgánicamente con la superficie:

- **Soft:** Elevación mínima para botones.
- **MD:** Sombra activa para tarjetas principales.
- **Premium:** Sombra de marca con alta dispersión para el FAB y elementos destacados.

---

## 🧪 Reglas Estéticas Finales

1. **Decoración Inteligente:** Los encabezados deben incluir formas orgánicas sutiles en los fondos.
2. **Interacción Táctil:** Cada toque debe generar una respuesta de Haptics y una transición de escala.
3. **Contraste Accesible:** Aunque los colores son pasteles, siempre mantenemos la legibilidad del texto principal sobre fondo blanco.
